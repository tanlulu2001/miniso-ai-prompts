## 每个模板标配四件套

| 套件 | 说明 |
|---|---|
| **Prompt 正文** | 带 `{{变量}}` 槽位，直接复制替换即可使用 |
| **变量说明表** | 每个变量填什么、示例值 |
| **常见坑与解法** | 翻车场景 + 一句话修复 |
| **真实运行示例** | 完整的输入 → 输出案例 |

---

## 设计原则

1. **变量化 > 具体化**：同一个模板换变量覆盖不同选题，不写死
2. **负面约束 > 正面约束**：告诉 AI 不能做什么比告诉它要做什么更有效
3. **角色 + 约束 + 输出格式 = 最小可用 Prompt**

---

## 使用方式

1. 找到对应场景的 Prompt 模板
2. 复制 Prompt 正文
3. 替换 `{{变量}}` 为实际内容
4. 粘贴到 Claude / ChatGPT 中执行
5. 如果翻车，翻到模板底部的"常见坑"查解法

---

## 维护规则

- 每次用坏了就改模板，改完记录版本号
- 有新的翻车场景就补充到"常见坑"
- 看到好的输入→输出案例就替换"真实示例"
- 版本号格式：v 主版本.次版本（v1.0 → v1.1 → v2.0）
